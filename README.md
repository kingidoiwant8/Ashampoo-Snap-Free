# Ashampoo Snap

## Overview

Ashampoo Snap is a screen capture and recording tool designed for Windows users. It enables capturing screenshots, recording screen activities, and editing captured content with ease. The software aims to provide a straightforward solution for creating visual documentation, tutorials, and presentations.

## Key Features

**Capture full screen, active windows, or user-defined areas**  
**Record screen video with or without audio input**  
**Edit captured images with annotation tools, text, and shapes**  
**Save outputs in multiple image and video formats**  
**Extract text from screenshots using OCR technology**  
**Automatic upload options for quick sharing**  
**Timed captures for delayed screenshot functionality**
